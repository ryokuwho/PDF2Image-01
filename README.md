# PDF to Image Converter - 仕様書 & 取扱説明書

## 1. アプリ概要
幼稚園や小学校から配布されるPDF形式のお手紙を、カレンダーアプリ（TimeTree等）に貼り付け可能なJPG画像に変換するためのWebアプリケーションです。

## 2. 開発の背景・目的
- **背景**: 学校からの連絡がPDFで届くが、多くのカレンダーアプリはPDFの直接添付に対応していない。
- **目的**: PDFを素早く高画質な画像に変換し、スマホのカレンダー予定に「画像として」貼り付けられるようにする。

## 3. 主な機能（要件）
### 変換機能
- **PDF読み込み**: ファイル選択またはドラッグ＆ドロップによるPDFの入力。
- **高画質画像化**: PDFの各ページを横幅1240ピクセル固定（縦は比率維持）のJPG画像に変換。
- **マルチページ対応**: 複数ページのPDFも全ページ個別に画像化。

### 保存・共有機能
- **個別保存**: 必要なページだけをJPGとしてダウンロード。
- **一括保存**: 全ページを連続してダウンロード（複数ページある場合に表示）。
- **クリップボードコピー**: 画像をクリップボードにコピーし、アプリ間で直接ペースト可能に。
- **OS標準共有 (Web Share API)**: スマホの共有メニューを呼び出し、LINE送信や写真ライブラリへの保存を容易にする。

### UI/UX
- **モバイルファースト**: スマホでの操作を前提とした大きなボタンと直感的なレイアウト。
- **進捗表示**: 変換中のローディングアニメーション。
- **プレビュー**: 変換後の画像を画面上で確認可能。

## 4. 技術仕様
- **フロントエンド**: React 19, TypeScript
- **スタイリング**: Tailwind CSS (v4)
- **アニメーション**: Framer Motion (motion/react)
- **PDF解析**: PDF.js (pdfjs-dist)
- **アイコン**: Lucide React
- **実行環境**: ブラウザ完結型（サーバーサイド処理なし）

## 5. セキュリティ・プライバシー
- **完全クライアントサイド処理**: PDFの解析および画像変換処理はすべてユーザーのブラウザ内で行われます。
- **データ非保持**: ファイルが外部サーバーにアップロードされることは一切ありません。

## 6. 使い方ガイド
1. **PDFの選択**: トップ画面の「PDFを選択」エリアをタップし、スマホ内のPDFファイルを選択します。
2. **変換の待機**: 「画像を生成中...」の表示が消えるまで待ちます。
3. **画像の確認**: 画面に表示されたプレビューを確認します。
4. **保存・共有**:
   - **スマホに保存したい場合**: 「保存」または「共有 ＞ 画像を保存」を選択。
   - **カレンダーにすぐ貼りたい場合**: 「コピー」を押してからカレンダーアプリでペースト、または「共有」からカレンダーアプリを選択。
   - **全部まとめて保存したい場合**: 上部の「全ページを一括保存」をタップ。

## 7. 今後の拡張性（メモ）
- 複数枚のPDFを1枚の長い画像に結合する機能。
- 画像のトリミング（必要な部分だけ切り出し）機能。
- OCR（文字認識）による予定の自動抽出。
